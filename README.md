# Instrucciones para clonar repositorio
Paso 1: Abrir GitHub Desktop

Si no lo tienes, puedes descargarlo desde la página oficial.

Paso 2: Clic en

File → Clone Repository…

Paso 3: Selecciona una de las dos opciones
A) Repositorio tuyo

Aparece en la lista → clic en Clone.

B) Repositorio de otra persona

Ve a la pestaña URL y pega el link del repo.
